How to Build a PC

  1. Choose a Processor (CPU)
    Pick a CPU first — it determines the socket and performance tier. Use "Select" on Processor to view available CPUs.
  2. Choose a Motherboard
    Select a motherboard compatible with the CPU socket and features (RAM slots, M.2, USB ports).
  3. Pick Memory (RAM)
    Choose RAM capacity and speed supported by the motherboard (eg. DDR4/DDR5).
  4. Graphics Card (GPU)
    If you game or do GPU work, pick a GPU that fits your budget and case size.
  5. Storage
    Select an SSD or HDD depending on speed and capacity needs. NVMe SSDs are fastest.
  6. Cooling & Power
    Pick cooling (air/liquid) and a PSU with enough wattage for your components.
  7. Review & Add to Cart
    After selecting items, review the build on the right and click "ADD TO CART" to save or purchase.
Tip: Start with the CPU and motherboard to ensure socket compatibility.
